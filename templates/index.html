NYC Restaurants

​ ​ ​

Overviews

​
​

The purpose of this project was to analyze restaurants inspection grades with different cuisines of New York City. To accomplish this analysis, we first pulled data from NYC Open Data and then built a flask API route for the dataset exported from MongoDB. This comprehensive DOHMH dataset includes restaurant name, contact information, address, borough, zip code, as well as cuisine type, food rating, food safety letter grade, inspection date and violations.This amount of data provide many options for interactive visualizations

​ ​

After assembling the dataset, we used D3 and Leaflet libraries from JavaScript to illustrate various aspects of the restaurants inspection scores/grades along with locations, cuisines options, etcs. Factors we looked at included: inspection grades, actual scores, locations and cuisines. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
